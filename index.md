# Политика конфиденциальности DuckonPro

**Дата вступления в силу:** 12 апреля 2026 г.

**Разработчик:** ITL Solutions  
**Контакт:** info@itlsolutions.net

---

## 1. Какие данные мы собираем

### Данные аккаунта
- Имя и фамилия
- Номер телефона (+992)
- Адрес электронной почты (необязательно)

### Данные магазина
- Название и адрес магазина
- Информация о товарах, продажах, расходах
- Данные о клиентах и поставщиках (имя, телефон)
- Данные о сотрудниках (имя, роль, рабочие часы)

### Данные устройства
- Модель устройства и версия ОС (для технической поддержки)
- Push-уведомления: токен устройства (Firebase Cloud Messaging)

### Данные Bluetooth
- Приложение использует Bluetooth для подключения к чековому принтеру
- Мы не собираем и не передаём данные Bluetooth-устройств

### Данные камеры
- Камера используется только для сканирования штрихкодов товаров
- Изображения с камеры не сохраняются и не передаются

### Данные о местоположении
- Приложение запрашивает доступ к местоположению только для работы Bluetooth (требование Android)
- Мы не отслеживаем и не сохраняем данные о местоположении

## 2. Как мы используем данные

- Для работы приложения (учёт товаров, продажи, отчёты)
- Для отправки чеков клиентам через Telegram
- Для отправки push-уведомлений о важных событиях (низкий остаток, завершение доставки)
- Для генерации отчётов и аналитики внутри приложения

## 3. Хранение данных

- Данные хранятся на защищённых серверах
- Пароли хешируются с использованием bcrypt
- API-запросы передаются по зашифрованному каналу (HTTPS)
- Токены аутентификации (JWT) хранятся в защищённом хранилище устройства (EncryptedSharedPreferences на Android)

## 4. Передача третьим лицам

Мы **НЕ** продаём, не передаём и не раскрываем ваши данные третьим лицам, за исключением:

- **Firebase (Google)** — для доставки push-уведомлений (только токен устройства)
- **Telegram Bot API** — для отправки чеков клиентам (только данные чека и chat ID)
- **Национальный банк Таджикистана (nbt.tj)** — для получения курсов валют (мы только читаем публичные данные, ничего не отправляем)

## 5. Безопасность

- Шифрование данных при передаче (TLS/HTTPS)
- Хеширование паролей (bcrypt)
- JWT-токены с ограниченным сроком действия
- Роли и права доступа (владелец, менеджер, кассир)
- Валидация данных на сервере

## 6. Ваши права

Вы имеете право:
- Запросить доступ к своим данным
- Исправить неточные данные
- Удалить свой аккаунт и все связанные данные
- Отказаться от push-уведомлений

Для реализации этих прав свяжитесь с нами: info@itlsolutions.net

## 7. Хранение и удаление

- Данные аккаунта хранятся пока аккаунт активен
- При удалении аккаунта все данные удаляются в течение 30 дней
- Резервные копии удаляются в течение 90 дней

## 8. Дети

Приложение не предназначено для лиц младше 18 лет. Мы не собираем данные несовершеннолетних.

## 9. Изменения

Мы можем обновлять эту политику. О существенных изменениях мы уведомим через push-уведомление или внутри приложения.
